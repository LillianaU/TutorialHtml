Bienvenido al Tutorial de HTML

HTML (HyperText Markup Language) es el lenguaje estándar para la creación de páginas web.

1. Estructura Básica

Un documento HTML tiene la siguiente estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página</title>
</head>
<body>
    <h1>Hola, Mundo</h1>
</body>
</html>

2. Etiquetas Comunes

Algunas etiquetas esenciales en HTML incluyen:

  • <p>: Para párrafos.
  • <a>: Para enlaces.
  • <img>: Para imágenes.
  • <ul> y <ol>: Para listas.
  • <table>: Para tablas.
  • <form>: Para formularios.
  • <div> y <span>: Para estructurar contenido.

3. Agregar Enlaces

Para agregar un enlace a otra página:

<a href="https://www.ejemplo.com" target="_blank">Visitar Ejemplo</a>

4. Agregar Imágenes

Para agregar una imagen a la página:

<img src="imagen.jpg" alt="Descripción de la imagen" width="300">

5. Creación de Tablas

Ejemplo de cómo crear una tabla en HTML:

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Juan</td>
        <td>25</td>
    </tr>
</table>

6. Formularios en HTML

Ejemplo de un formulario simple:

<form action="submit.php" method="POST">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>
    <input type="submit" value="Enviar">
</form>

7. Ejercicios de Práctica

Realiza los siguientes ejercicios para reforzar tu conocimiento en HTML:

  1. Crea una página HTML con un encabezado <h1> y un párrafo <p>.
  2. Agrega un enlace que lleve a "https://www.google.com".
  3. Inserta una imagen con la etiqueta <img> y usa un texto alternativo descriptivo.
  4. Crea una lista ordenada (<ol>) con tres elementos.
  5. Crea una tabla con tres columnas: Nombre, Edad y Ciudad.
  6. Agrega un formulario con un campo de texto y un botón de enviar.
  7. Usa <div> para estructurar un contenido con un título y una imagen dentro.
  8. Agrega un botón con <button> y un mensaje dentro.
  9. Usa la etiqueta <span> para resaltar una palabra dentro de un párrafo.
  10. Incluye un comentario en tu código HTML explicando su propósito.
